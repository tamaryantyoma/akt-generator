## 📖 О проекте

**AKT Generator** — это приложение, разработанное для автоматизации оформления актов приема-передачи техники.

Программа заполняет готовый шаблон Word, подставляя данные сотрудника и оборудования, после чего сохраняет полностью готовый документ без необходимости ручного редактирования.

Проект создавался для внутреннего использования в IT-отделе компании и позволяет сократить время оформления документов с нескольких минут до нескольких секунд.

---

## ✨ Возможности

* 📄 Генерация актов в формате **Microsoft Word (.docx)**
* 🌍 Двуязычные документы (русский + армянский)
* 👤 Автоматическое заполнение ФИО
* 🧑 Поддержка мужского и женского рода
* 🌐 Выбор гражданства
* 💻 Заполнение модели ноутбука
* 🔢 Заполнение серийного номера
* 📅 Автоматическая подстановка текущей даты
* 📁 Автоматическое создание папки для каждого сотрудника
* 💾 Запоминание моделей ноутбуков в JSON
* 🌐 Веб-интерфейс на Flask
* 🖥️ Возможность запуска как локального приложения

---

## 🖼️ Интерфейс

В веб-интерфейсе пользователь вводит:

* ФИО на армянском языке
* ФИО на русском языке
* пол сотрудника
* гражданство
* модель ноутбука
* серийный номер

После нажатия кнопки **"Сгенерировать"** приложение создает готовый документ Word.

## 📄 Как работает генерация

Программа:

1. Загружает шаблон Word.
2. Разбирает введённые ФИО.
3. Автоматически определяет текущую дату.
4. Подставляет значения в нужные поля шаблона.
5. Заполняет:

   * модель ноутбука;
   * серийный номер;
   * гражданство;
   * пол сотрудника.
6. Создает папку с фамилией сотрудника и текущей датой.
7. Сохраняет готовый документ.

## 💾 Список моделей

Модели ноутбуков сохраняются в файле

```text
laptop_models.json
```

Новые модели автоматически добавляются после первой генерации документа и затем доступны для повторного использования.

## 📌 API

### Получить список моделей

```
GET /api/models
```

---

### Добавить модель

```
POST /api/models
```

JSON

```json
{
  "model": "Dell Latitude 5450"
}
```

---

### Генерация документа

```
POST /api/generate
```

Пример запроса

```json
{
  "name_arm":"Արմեն Պետրոսի Սարգսյան",
  "name_rus":"Саргсян Армен Петросович",
  "gender":"Мужской",
  "citizenship":"РА",
  "model":"Dell Latitude 5450",
  "serial":"ABC123456"
}
```
